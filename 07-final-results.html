Key Architectural Novelties

Kebaruan Arsitektural Utama

  1. Saturation-awareness for Mw > 6.5: Kesadaran saturasi untuk Mw > 6,5: classical τc/Pd features saturate above magnitude 6.5 due to the corner-frequency drop below the post-P window bandwidth. IDA-PTW bypasses this by routing large-event prediction through the intensity gate rather than magnitude estimation. fitur klasik τc/Pd jenuh di atas magnitudo 6,5 karena penurunan corner-frequency di bawah bandwidth window pasca-P. IDA-PTW melewati ini dengan merouting prediksi event besar melalui intensity gate, bukan estimasi magnitudo.
  2. Intensity-driven adaptive PTW: PTW adaptif intensity-driven: window size {3, 5, 8}s selected per intensity class — short window for damaging events (where speed is critical), long window for non-damaging (where accuracy can be improved without operational cost). ukuran window {3, 5, 8}s dipilih per kelas intensitas — window pendek untuk event damaging (di mana kecepatan kritis), window panjang untuk non-damaging (di mana akurasi dapat ditingkatkan tanpa cost operasional).
  3. Catalog-independent operation: Operasi catalog-independent: the system makes no magnitude estimate and requires no event catalog query — Sa(T) is predicted directly from the 0.5 s P-wave window for Stage 0, 3–8 s adaptive window for Stage 2. sistem tidak membuat estimasi magnitudo dan tidak memerlukan query katalog event — Sa(T) diprediksi langsung dari window P-wave 0,5 s untuk Stage 0, window adaptif 3–8 s untuk Stage 2.
  4. 103-period Sa(T) prediction: Prediksi Sa(T) 103-periode: the full spectral acceleration response curve T ∈ [0.01, 10] s is regressed simultaneously, enabling targeted alerting per asset-class (see § 03 Derivations for the Sa(T) ↔ building-height mapping). kurva respons spectral acceleration penuh T ∈ [0,01, 10] s di-regresi simultan, memungkinkan alerting tertarget per kelas aset (lihat § 03 Derivasi untuk pemetaan Sa(T) ↔ tinggi gedung).
